### Installation (Termux)
You can easily install **Srsu** in Termux by using the command below
```
termux-setup-storage
apt update
apt upgrade
pkg install wget
pkg install php
pkg install curl
pkg install git
```
Just Clone this repository
  ```
git clone --depth=1 https://github.com/ultra303/srsu.git
```
Just continue to run **Srsu**
```
cd srsu
chmod +x srsu.sh
bash srsu.sh
```
